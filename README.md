# Picker Performance Dashboard Sample

Static live sample for the warehouse picker performance board proposal.

Open `index.html` in a browser. No install or build step is required.

Files:

- `spec.md` - project spec
- `index.html` - dashboard page
- `styles.css` - visual system
- `app.js` - sample data and timeline controls
- `assets/picker-board-proposal.png` - proposal image reference
